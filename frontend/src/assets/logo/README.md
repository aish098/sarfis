# Logo Assets Folder

Place your corporate logo files in this directory. 

### Recommended Formats:
* **Vector Logo:** `logo.svg` (Highly recommended for high-DPI and responsiveness)
* **Raster Logo:** `logo.png` (Transparent background, minimum resolution: `128x128px`)

Once you place your logo file here, we can update the `Navbar.jsx` branding element to render your custom logo image instead of the default icon.
